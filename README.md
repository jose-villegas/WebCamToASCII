WebCamToASCII
=============

Proyecto de Procesamiento Digital de Imagenes

Este programa toma imagenes directamente desde la camara web u otro dispositivo de captura de imagen y convierte la imagen en ASCII
art en tiempo real.

Dependencias: AForge.
